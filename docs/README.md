# GitHub Pages Site

`index.html` is a self-contained static dashboard. Supporting CSV downloads are under `data/`.

The workflow `.github/workflows/pages.yml` publishes this directory through GitHub Pages.

After pushing the repository:

1. Open repository **Settings**.
2. Open **Pages**.
3. Select **GitHub Actions** as the source.
4. Run the Pages workflow or push to `main`.

The dashboard disclaimer must remain visible in all derivative versions.
