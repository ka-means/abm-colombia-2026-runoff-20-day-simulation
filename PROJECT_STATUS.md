# Project Status

## Current release

Version `1.0.0` packages the completed twenty-day heuristic ABM, the completed post-ABM Claude deliberation outputs supplied with the project, and the interactive analytical dashboard.

## Stable components

- audited source package v5.1;
- source-category balancing and event-cluster contracts;
- 1,000-agent full-run configuration;
- twenty-day ABM schedule;
- cognitive-dossier and structured-decision architecture;
- dashboard HTML and the supplied analytical tables;
- release history package with agent-level records.

## Components requiring further engineering

- migration of stable notebook functions into importable Python modules;
- full dependency lock generated from a clean successful runtime;
- automated parameter sensitivity experiments;
- multiple-seed Monte Carlo analysis;
- systematic prompt/model robustness analysis;
- regeneration of the dashboard tables listed as missing in `section_23_dashboard_manifest.csv`;
- external replication by an independent environment.

## Interpretation boundary

Completion of the software pipeline does not establish empirical validity for real electoral prediction. The next scientific step is validation and sensitivity analysis, not stronger forecasting language.
