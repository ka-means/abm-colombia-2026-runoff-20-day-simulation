# Results

This directory contains supplied analytical tables and lightweight examples supporting the static dashboard.

## `dashboard_tables/`

Contains the CSV files received with the project plus `section_23_dashboard_manifest.csv`. The manifest distinguishes included tables from tables that the notebook can regenerate but that were not present in the uploaded file set.

## `samples/`

Contains small, review-friendly subsets. Samples are not substitutes for the full result tables.

## Interpretation

Every table describes synthetic model behaviour. Regional, demographic, emotional, linguistic and candidate-state summaries are internal simulation outputs.

Large raw logs, full cognitive dossiers, GraphML networks and the executed notebook belong in a versioned GitHub Release rather than ordinary Git history.
