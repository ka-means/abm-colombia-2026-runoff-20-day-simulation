# Scripts

- `validate_data_package.py`: validates the audited ZIP and master-table contracts.
- `validate_results.py`: validates supplied result-table dimensions, labels and headline KPI consistency.
- `publish_dashboard.py`: copies a dashboard HTML file into `docs/index.html`.
- `generate_checksums.py`: creates a SHA-256 manifest for a directory tree.

All scripts are safe to run without an API key.
